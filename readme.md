# Kebbet plugins - Custom post types and taxonomies: betshammar.se
- Adds a custom post type: `project`
- Adds a custom post taxonomy: `artyear` for post type `project`.
- Adds a custom post taxonomy: `arttype` for post type `project`.
- Adds a custom post taxonomy: `artstatus` for post type `project`.

## How to use
- Activate plugin and use with normal WordPress UI.
- Use theme to get content.

## Dependencies
- Advanced Custom Fields Pro (for plural labels on `arttype`).
